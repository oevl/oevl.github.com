All the cool kids are using Git

Published 2nd January 2008

In the past week i switched all of my local Subversion repositories to Git, i made this change after reading Version Control and “the 80%” - which ironically was written by a Subversion developer.

After reading the article and watching Linus Torvald's Google Talk about Git, i took the Git - SVN Crash Course and was immediately hooked.

I will not write about all the cool stuff Git can do or outline the differences with Subversion because others have already done this very good.
