Great stuff

Published 24th July 2008

1Password
Password manager which lets you auto-fill login forms from the web browser.
Caffeine
Prevents the Mac from going to sleep. Great when watching a flash video or reading stuff.
Flow
FTP/SFTP application with a great UI.
App Engine Launcher for Mac OS X
Really helps managing App Engine applications.
HandBrake
Simplest ripping application I've seen for OS X. I especially like the presets.
IPython
Enhanced Python shell with a lot of time-saving features.
Liberation Mono Font
After trying out various programming fonts (including Consolas, Monaco, ProFont and Proggy) I'm pretty happy with this one. Peter Cooper has some screenshots.
Limechat
Simple IRC client written in RubyCocoa.
PEP8.py
Script which checks Python source code against some of the style conventions described in PEP 8.
PyFlakes
Checks Python source code for common errors such as unused or missing imports, without executing it.
Safari AdBlock
Similar to the Firefox AdBlock addon.
Terminal.app IR_Black Theme
Dark theme which looks a lot better than the ones that come with Terminal.app.
